Start Up has a number of different layouts. It can be a fixed or fluid width and it can have a sidebar or be full width. You can combine these in many ways. This is done easily by adding/changing a stylesheet. The layout uses the default 12 column, fluid layout that generally comes with the bootstrap default css file.

Start Up also scales down automatically to mobile devices of different resolutions. It works perfectly on an iPhone for example (without changing the HTML of the page).

Important - If you would like to change the theme go to main.scss and change the variables and compile it.

This template was built using Twitter Bootstrap - which means you have access to a wide variety of built in CSS from the main.scss file (that has already been tested across major browsers by the Twitter Bootstrap group).

If you would like to edit a specific section of the site, simply find the appropriate label in the Scss file, and then scroll down until you find the appropriate style that needs to be edited.
